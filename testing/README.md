# Testing — JUnit 5 & Mockito

## Topics to Practice
- [ ] JUnit 5 basics — `@Test`, `@BeforeEach`, `@AfterEach`, `@BeforeAll`, `@AfterAll`
- [ ] Assertions — `assertEquals`, `assertThrows`, `assertAll`, `assertTimeout`
- [ ] `@ParameterizedTest` — `@ValueSource`, `@CsvSource`, `@MethodSource`
- [ ] `@Nested` test classes
- [ ] `@DisplayName`, `@Tag`, `@Disabled`
- [ ] `@TempDir`, `@ExtendWith`
- [ ] Mockito — `@Mock`, `@InjectMocks`, `@Spy`
- [ ] Stubbing — `when().thenReturn()`, `when().thenThrow()`
- [ ] Verification — `verify()`, `times()`, `never()`, `ArgumentCaptor`
- [ ] `doReturn()` vs `when()` for spies
- [ ] Mocking static methods (`mockStatic`)
- [ ] TDD cycle — Red, Green, Refactor
- [ ] Test doubles — dummy, stub, fake, spy, mock
- [ ] Testing concurrency
- [ ] Testing exceptions
