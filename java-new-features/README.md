# Java New Features

## Java 8 (`java8/`)
- [ ] Lambdas & functional interfaces
- [ ] Streams API
- [ ] `Optional`
- [ ] Default & static interface methods
- [ ] Date/Time API
- [ ] `CompletableFuture`
- [ ] Nashorn JS engine (removed in 15)

## Java 9 (`java9/`)
- [ ] Module system (JPMS)
- [ ] `List.of()`, `Set.of()`, `Map.of()`
- [ ] `Stream.takeWhile()`, `dropWhile()`, `iterate()` with predicate
- [ ] `Optional.ifPresentOrElse()`, `stream()`
- [ ] Private interface methods

## Java 10 (`java10/`)
- [ ] `var` — local variable type inference
- [ ] `List.copyOf()`, `Map.copyOf()`

## Java 14 (`java14/`)
- [ ] Records (preview)
- [ ] `switch` expressions (stable)
- [ ] `instanceof` pattern matching (preview)
- [ ] Helpful NullPointerExceptions

## Java 16 (`java16/`)
- [ ] Records (stable)
- [ ] `instanceof` pattern matching (stable)
- [ ] Stream `toList()`

## Java 17 (`java17/`)
- [ ] Sealed classes & interfaces
- [ ] Text blocks (stable)
- [ ] Pattern matching improvements

## Java 21 (`java21/`)
- [ ] Virtual threads — `Thread.ofVirtual()`
- [ ] Structured concurrency
- [ ] Sequenced collections — `SequencedCollection`, `SequencedMap`
- [ ] Pattern matching in `switch` (stable)
- [ ] Record patterns
- [ ] String templates (preview)
